Weather Analysis Dashboard using Power BI
📌 Project Overview

The Weather Analysis Dashboard is an interactive Power BI project developed to analyze and visualize weather data across multiple districts. The dashboard provides valuable insights into rainfall, temperature, humidity, and wind speed using dynamic visualizations, KPI cards, DAX measures, and filters. It enables users to identify seasonal patterns, compare weather conditions between districts, and detect unusual weather events through an intuitive and interactive interface.

🎯 Objectives
Analyze rainfall, temperature, humidity, and wind speed.
Compare weather metrics across districts and mandals.
Identify seasonal weather trends and yearly patterns.
Detect extreme weather events and anomalies.
Build an interactive dashboard for data-driven decision making.
